# 피그마 별 모양 사용 방법

## 방법 1: SVG로 내보내기 (권장)

1. 피그마에서 별 모양을 선택합니다
2. 우클릭 → **Copy as SVG** 또는 **Export** → SVG 선택
3. SVG 코드를 복사하거나 파일로 저장합니다

### SVG 파일로 저장하는 경우:
1. 피그마에서 별 모양을 선택
2. 우측 패널에서 **Export** 클릭
3. **SVG** 형식 선택 후 **Export** 버튼 클릭
4. 다운로드된 SVG 파일을 프로젝트의 `images` 폴더에 저장 (예: `star.svg`)

### SVG 코드를 직접 사용하는 경우:
1. 피그마에서 별 모양 선택
2. 우클릭 → **Copy as SVG**
3. `script.js`의 `renderStars` 함수에서 SVG 코드를 사용하도록 수정

## 방법 2: PNG/이미지로 내보내기

1. 피그마에서 별 모양 선택
2. **Export** → **PNG** 또는 **JPG** 선택
3. 해상도 설정 (예: 2x 또는 3x for retina displays)
4. 다운로드된 이미지를 `images` 폴더에 저장
5. CSS에서 `background-image`로 사용

## 현재 구현 방식

현재는 CSS `clip-path`를 사용하여 별 모양을 만들고 있습니다.
피그마에서 만든 별 모양을 사용하려면 위의 방법 중 하나를 선택하여 적용하시면 됩니다.
